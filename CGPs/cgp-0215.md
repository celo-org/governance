## Overview

This proposal requests $22,000 (183,333 CELO) to settle outstanding bug bounty rewards, concluding the administrative phase of the Celo SHIELD program.

### Status

- DRAFT

### Proposal Description

The Celo SHIELD (Subsidized Help for Improving Ecosystem-Level Defense) program successfully provided security services to projects building on Celo throughout 2025. As part of the program's competitive bug bounty component, security researchers identified and reported vulnerabilities that improved the security posture of the ecosystem.

This proposal represents the final administrative action for the Celo SHIELD programâ€”settling outstanding payments owed to security researchers who contributed valid vulnerability reports during the program's operation.

## Background

The original Celo SHIELD program was funded through CGP-157 (225,000 cUSD) with the goal of subsidizing audit and security costs for projects building on Celo. The program included a competitive bug bounty component with a budget of 60,000 cUSD allocated specifically for rewarding security researchers who identify vulnerabilities.

Bug bounty programs are critical for:
- Incentivizing responsible vulnerability disclosure
- Identifying security issues before malicious actors can exploit them
- Building trust with the security research community
- Maintaining Celo's reputation as a secure platform

Honoring commitments to security researchers is essential for maintaining Celo's credibility and encouraging future responsible disclosures to the ecosystem.
